夜空下的遥想

发表于 2017-03-31 | 分类于 文字的力量

午夜子时，却如何也无法入眠，推开了屋门，一阵冷寂猛然袭来，黑夜吞噬了整个世界，唯有一轮苍白的孤月镶嵌在无边的黑色夜空，刚才还万家灯火的人们现在已悄然入睡，虫鸣声给这寂静的夜又添了几分安详，也许是我长大了，面对这漆黑的夜空感觉不到一丝害怕，猛然间，我感到了生命的存在，更感到了生命的流逝，这黑夜，触动了我的心扉之门。

从呱呱坠地那一刻，我们与这个世界相约，在父母的无私呵护下，我们渐渐的，从躺着到爬行，从爬行到直立，从直立到奔跑！

有一天，我们被送进了学校，从开始的不适应，慢慢的，我们忘了这种感觉，这时的我们，没有害羞，没有张扬，更没有尔虞我诈和妄自尊大，我们散发着人类的本性之美，我们，有时哭，有时闹，但更多的是笑笑逐颜开，上课后我们：a o e.下课后我们：捉迷藏，伴着对童真与对未知的好奇，我们步入初中，这时的我们，有的可能离开了家乡，独在异乡，难免会思乡，可这毕竟最后被时间冲淡，我们在稍微高级的学校中逐渐走向“规定”，我们按时起床，准时睡觉，定点吃饭，慢慢的，我们养成了不是习惯的习惯，而初三那一年，我们又几乎忘了吃饭，不想睡觉，生怕时间溜走，在老师的指引下，我们对未来充满了无尽幻想，抱着对理想的追求与渴望，我们咬牙坚持着，可，现在的我们，仍然只是孩子，终于，最后的最后，生活给了每个人一份打卷，我们得到的结果不同，但，我们依然有一颗充满血色的心，拿着张张红纸，我们走近了高中——一个初中时，我们幻想的“梦幻天堂”，之后才发现，这才是真正的“人间地狱”，突然想到了苏轼说的一句话：古之成大事者，非唯有超世之才，亦必有坚韧不拔之志，从此，我把它当作座右铭，去奋斗，去拼搏，可中间，我走错了许多，也许是对未知事物的好奇吧，可贵的是，迷途而知反，也是这些经历，更让我有了更深的人生感悟，高中一句话：痛并快乐着。突然想说：其实，快乐与痛苦并没有分界线。

一条幽径，香花弥漫中顿显醉人的那个妩媚，然曲折迂回中亦能激起心旷神亦的向往，挫折面前，抬起头，别让我们12年的努力戛然而止，抬起头，change yourself!

我们逐渐走向成熟，也注定背负更多，别忘了，在闲暇时刻，去找一下小时候的自己，不要在意他人怎么想，然后一专一的态度，踏实的心态，无比的从容，坚韧的毅力，永不言败！对我真诚的人，我会记得，欺骗我的人，我会淡忘，珍惜拥有的，忘记该忘的，人这一生不太长，何必给自己太多烦恼！

月升，月落：一夜；日升，日落：一天；睁眼，闭眼：一生！
